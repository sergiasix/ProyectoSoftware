# RailTravel 🚆

Proyecto académico de ingeniería del software para la planificación y desarrollo de una plataforma de gestión de billetes de tren.

## Estado del proyecto
⚠️ Proyecto en desarrollo (no finalizado)

Actualmente implementado:
- Búsqueda de trenes
- Visualización de resultados
- Selección de asiento

Pendiente:
- Persistencia de datos
- Integración con APIs reales
- Gestión completa de reservas

## Tecnologías
- Backend: Spring Boot (Java)
- Frontend: HTML, CSS, JavaScript
- Base de datos: No implementada (simulada)

## Decisiones técnicas
Se ha optado por simular datos en lugar de usar APIs reales debido a:
- Complejidad de integración
- Limitaciones de tiempo
- APIs no públicas

## Deuda técnica
- Falta de validaciones
- Código frontend no modular
- Sin persistencia de datos
- Lógica simplificada de reservas

## Ejecución
1. Ejecutar backend (Spring Boot)
2. Abrir index.html en navegador
3. Realizar búsqueda
